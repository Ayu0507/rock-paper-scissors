# Rock Paper Scissors Game 

A simple Rock Paper Scissors game built using **HTML5** and **JavaScript**. The player competes against the computer, which randomly chooses **Rock**, **Paper**, or **Scissors**. The game instantly displays whether the player wins, loses, or ties.

## Features

* Play against the computer
* Random computer move using `Math.random()`
* Instant game result
* Interactive buttons
* Beginner-friendly JavaScript project

## Built With

* HTML5
* JavaScript
